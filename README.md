# Triangle Type Checker in C

## About Project
This mini project is written in C language.
It checks whether a triangle is valid or not and identifies its type.

## Features
- Valid triangle check
- Equilateral triangle detection
- Isosceles triangle detection
- Scalene triangle detection

## Concepts Used
- if-else statements
- Nested if
- Logical operators
- User input

## Triangle Rules

For a valid triangle:
- a + b > c
- a + c > b
- b + c > a

## Triangle Types

### Equilateral Triangle
All sides are equal.

Example:
3 3 3

### Isosceles Triangle
Any two sides are equal.

Example:
5 5 3

### Scalene Triangle
All sides are different.

Example:
3 4 5
